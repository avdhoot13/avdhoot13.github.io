Multimedia Content Security System

1. MCSS

1) Our Project represents a double stage encryption algorithm that provides the security of multimedia contents such as image, audio and video in the cloud.
2) The proposed algorithm is crucial in the second stage. The randomly generated key provides more security than the conventional encryption system.
3) The 94-bit converter generates the multimedia contents into the ciphertext. The ciphertext is stored in the cloud instead of original multimedia content.

2. Random Key Generation

1) The randomly generated key is unknown to the content manager too.
2) Without the randomly generated key, the Decryption-I process is difficult and thus the proposed architecture gives efficient security.
3) The Encryption-2 and the Decryption-I process is based on the random key generation shown by the origin color in the system server and the client

3. Sharing Files

1. Share module in Cloud helps in sharing of resources dynamically that comprises of Computation, Storage and Software
2. It helps the user to easily share their file whether it is doc file, image file, pdf file, video file, etc. with those user who are within the cloud.
